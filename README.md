# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built using **HTML, CSS, and JavaScript**.  
This project is designed for college students and beginners to showcase their skills, projects, resume, and contact information.

---

# 📌 Features

✅ Responsive Design  
✅ Modern UI/UX  
✅ Home Section  
✅ About Section  
✅ Skills Section  
✅ Projects Showcase  
✅ Contact Form  
✅ Mobile Navigation Menu  
✅ Smooth Hover Effects  
✅ Clean and Simple Code Structure  

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
